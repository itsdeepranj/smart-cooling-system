# Operating Modes

## Non-operation Modes

### OFF MODE

    1. Controller powerd down
    2. Pump Off
    3. Fan Off
    4. Display Off

### Intialization Mode (Key Position 1)

    1. Controllers boots
    2. All sensors are read
    3. Sensors diagnostics performed
    4. Fault condiiton detected
    5. Display active
    6. Pump Off
    7. Fan Off

## Normal Operation

### Run Mode (Key Position 2)

    1. Cooling control algoritham active
    2. Pump controlled based on temperature
    3. Fan controlled based on temperature
    4. Display shows the system status
    5. Fault monitoring active
